# Tallynproject
Price Comparison Website
Welcome to our Price Comparison Website project! This web application is designed to help users compare prices of products between two major e-commerce platforms: Amazon and Flipkart. Our goal is to provide a simple, yet powerful tool to ensure you get the best deals while shopping online.

Features
Price Comparison: Easily compare prices of the same product between Amazon and Flipkart.
User-Friendly Interface: A clean, intuitive interface designed in Figma, ensuring a seamless user experience.
Responsive Design: Fully responsive web design that looks great on both desktop and mobile devices.
Bookmark Products: Users can bookmark products to revisit them later.
Price Alerts: Get notified when the price of a bookmarked product drops.
Technologies
This project is built with:

HTML5: For structuring the web content.
CSS3: For styling the website, ensuring it is visually appealing and responsive.
JavaScript: For adding interactive elements to the website, enabling dynamic content updates without needing to reload the page.
Figma: Used for designing the basic UI/UX before moving to code.
